# Find My Mahj Game

Website for **Find My Mahj Game** — a national directory connecting mahjong players with local groups, venues, retreats, tournaments, and events.

**Live site:** [findmymahjgame.com](https://findmymahjgame.com)

## About

Find My Mahj Game serves 600,000–750,000 American mahjong players by helping them find:

- Local players and groups
- Venues and "Official Mahj Spots"
- Retreats and tournaments
- Open play events

## Sister Site

[Las Vegas Mahjong](https://lasvegasmahj.com) — mahjong lessons, events, and open play in Las Vegas. ([GitHub repo](https://github.com/lasvegasmahj-maker/lasvegasmahj))

## Tech Stack

- Next.js 16 (React, TypeScript)
- Tailwind CSS 4 + shadcn/ui
- Supabase (database + auth)
- Vercel (hosting, auto-deploy from GitHub)

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy

Pushes to `main` auto-deploy via Vercel.
